## Standard SaaS marketing video structure (NeuraFlow-style)

1. **Hook / pain point** — bold kinetic typography stating the problem, fast cuts
2. **Product reveal** — app UI slides/zooms in, often inside device frame
3. **Feature montage** — 3–4 rapid feature scenes, each: UI close-up + short benefit label
4. **Proof / outcome** — metric, progression, or social proof moment
5. **CTA** — logo lockup, tagline, button animation, URL

## Style conventions
- High-contrast brand color on dark or light background
- Spring/ease-out entrances; elements never linger — every 2–4s something moves
- Text animates per-word or per-line (slide-up + fade)
- UI mockups float with soft shadows, slight 3D tilt or parallax
- Beat-synced cuts (no audio in v1 — keep cuts at even intervals)
